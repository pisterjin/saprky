# 자격 증명 변경 방법

1. Windows 검색에서 "자격 증명 관리자" 검색 후 실행
2. Windows 자격 증명 탭 클릭
3. 목록에서 git:https://github.com 찾기
4. 클릭 후 제거
5. 다시 git push -u origin main 실행
6. GitHub 로그인 창 뜨면 pisterjin 계정으로 로그인

이렇게 하시면 됩니다.

저장소 구조나 연결은 그대로 유지됩니다.

답변)
자격 증명 관리자 삭제와 별개로, git에 설정된 사용자 이름이 따로 있습니다. 
커밋 작성자 정보는 아래 명령어로 설정된 값을 씁니다.

git config --global user.name
git config --global user.email

위 명령어로 현재 설정된 값을 확인해보시고, 
jeongjinhwa7 관련 정보가 나오면 아래처럼 변경하시면 됩니다.

git config --global user.name "pisterjin"
git config --global user.email "pisterjin 이메일"
단, 이미 올라간 커밋의 작성자 표시는 바뀌지 않고 이후 커밋부터 반영됩니다.

터미널 없이 간단한 파일을 올릴 때 쓰는 기능입니다. 
다만 여러 파일을 한꺼번에 올리거나 코드 작업을 할 때는 터미널로 push하는 게 더 편합니다.

Create new file → GitHub 웹에서 직접 파일을 작성해서 올리는 기능
GitHub 웹사이트 안에서 직접 코딩/타이핑해서 새 파일을 만드는 기능입니다. 
로컬 파일 선택이 아니라 웹에서 직접 작성하는 것입니다.

Upload files → 로컬 파일을 드래그앤드롭이나 선택으로 업로드하는 기능
로컬 파일을 선택하거나 드래그앤드롭으로 올리는 기능입니다. 
복수(files)인 이유는 말씀하신 대로 여러 파일을 한꺼번에 올릴 수 있어서입니다.
